# MCP Fetch Research

## Purpose
Gather and structure web evidence through MCP fetch server workflows.

## Instructions
1. Select 2-3 target URLs aligned with research question.
2. Fetch content via MCP fetch.
3. Extract key claims and supporting snippets.
4. Output per claim: source, date, confidence (HIGH/MEDIUM/LOW), relevant quote.
5. Summarize implications and open questions.

## Constraints
Do not report unsupported claims.
Keep source attribution explicit.
Mark low confidence clearly.
