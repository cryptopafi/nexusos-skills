# Affiliate Campaign

## Purpose
Create launch-ready affiliate campaign assets and measurement structure.

## Instructions
1. Define offer, target segment, and campaign objective.
2. Draft SMS copy (<=160 chars) and landing page primary copy.
3. Produce tracking link schema and source tags.
4. Add KPI template: CTR, conversion rate, EPC, revenue per click.
5. Return final package in implementation order.

## Constraints
No unverified claims in ad copy.
No campaign without explicit target/audienc
