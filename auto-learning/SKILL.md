# Auto Learning

## Purpose
Transforma surprizele si greselile in lectii persistente reutilizabile.

## Instructions
1. Dupa task complex, intreaba: a existat o surpriza reala?
2. Daca nu, skip rapid fara zgomot.
3. Daca da, extrage root cause: eroare personala sau gap sistemic.
4. Pentru eroare personala, salveaza in Cortex cu `trigger`, `mistake`, `fix`, `rule_ref`, tag-uri `lesson`, `auto-learning`.
5. Pentru gap sistemic, adauga propunere in `memory/rules/rules-candidates.md`.
