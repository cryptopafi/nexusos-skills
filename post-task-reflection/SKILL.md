# Post-Task Reflection

## Purpose
Extract reusable lessons after meaningful work and store them in a searchable format.

## When To Use
Run reflection after:
- Any task taking more than `30 minutes`
- Any task with output over `2,000 tokens`
- Any task with error/rework
- Any task with new discovery or unusually strong results

Skip only trivial reads/formatting/lookups.
