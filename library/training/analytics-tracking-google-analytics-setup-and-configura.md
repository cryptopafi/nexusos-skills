# Google Analytics Setup and Configuration

[ANALYTICS-TRACKING] Google Analytics Setup and Configuration — Procedura Standard de Operare
Scope: Configurare completă proprietate GA4 — de la creare până la linking cu Google Ads și Search Console, cu events și conversii active.
Steps:
- Creează proprietate GA4
- Instalează codul de tracking
- Configurează Data Streams
- Configurează events de bază
- Creează conversion events
- Linkuiește cu Google Ads
- Linkuiește cu Search Console
- Configurează rapoarte de bază
