# Apply Few-Shot Examples with Chain-of-Thought

[AI-PROMPT-ENGINEERING] Apply Few-Shot Examples with Chain-of-Thought — Procedura Standard de Operare
Scope: Combines multiple worked examples (few-shot) with explicit step-by-step reasoning traces (chain-of-thought) in a single prompt to teach the AI both the correct output format and the reasoning process required to produce it.
Steps:
- Identify and Characterize the Target Task
- Create 2-3 High-Quality Worked Examples
- Sequence Examples from Simple to Complex
- Construct the Full Prompt
- Evaluate Whether the AI Followed the Reasoning Pattern
- Optimize Example Count and Quality
