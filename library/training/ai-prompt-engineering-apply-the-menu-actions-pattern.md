# Apply the Menu Actions Pattern

[AI-PROMPT-ENGINEERING] Apply the Menu Actions Pattern — Procedura Standard de Operare
Scope: Configures the AI to present a structured menu of available actions at decision points, allowing the user to navigate complex interactions by selecting from pre-defined options rather than constructing new prompts from scratch.
Steps:
- Define the Context and Available Actions
- Instruct the AI to Present a Menu After Each Response
- Execute Initial Request and Review the First Menu
- Navigate the Interaction by Selecting Menu Items
- Handle Custom Actions When Needed
- Extract and Document the Interaction Path
