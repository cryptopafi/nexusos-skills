# Build a Diversified Crypto Portfolio

[CRYPTO] Build a Diversified Crypto Portfolio — Procedura Standard de Operare
Scope: Construirea unui portofoliu crypto diversificat pe categorii de active pentru reducerea riscului de concentrare.
Steps:
- Stabilește bugetul total și profilul de risc
- Definește categoriile de alocare
- Selectează activele per categorie (minim 5 total)
- Calculează și verifică alocările procentuale
- Execută achizițiile gradual (DCA)
- Configurează alerte de rebalansare
- Monitorizează și revizuiește lunar
