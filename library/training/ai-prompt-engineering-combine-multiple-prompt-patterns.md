# Combine Multiple Prompt Patterns

[AI-PROMPT-ENGINEERING] Combine Multiple Prompt Patterns — Procedura Standard de Operare
Scope: Selects, combines, and tests 2-3 complementary prompt patterns in a single interaction to address complex tasks that no single pattern can handle adequately, while managing pattern interference and optimizing combined effectiveness.
Steps:
- Analyze the Task Requirements by Dimension
- Select 2-3 Complementary Patterns
- Design the Combined Prompt Architecture
- Run a Minimal Test First
- Resolve Interference Issues
- Execute Full Task and Document the Pattern Stack
- Build a Reusable Pattern Stack Library
