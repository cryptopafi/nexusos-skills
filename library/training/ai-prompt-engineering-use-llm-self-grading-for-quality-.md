# Use LLM Self-Grading for Quality Assurance

[AI-PROMPT-ENGINEERING] Use LLM Self-Grading for Quality Assurance — Procedura Standard de Operare
Scope: Uses the AI to evaluate the quality of its own outputs against defined criteria using a structured scoring rubric, creating an automated quality assurance loop that catches low-quality outputs before they are used or distributed.
Steps:
- Define the Grading Rubric Before Generating Content
- Generate the Initial Output
- Request a Structured Self-Grade
- Review the Grade and Reasoning
- Request Targeted Improvements for Low Scores
- Re-Grade After Improvements
