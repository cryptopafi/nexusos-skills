# Apply the Audience Persona Pattern

[AI-PROMPT-ENGINEERING] Apply the Audience Persona Pattern — Procedura Standard de Operare
Scope: Configures the AI to tailor all outputs — language, tone, depth, format, and examples — to fit a precisely defined target audience persona rather than producing generic content.
Steps:
- Define the Audience Persona in Detail
- Embed the Audience Persona in the Prompt
- Specify Format Preferences Tied to Audience
- Generate Content and Inspect Calibration Statement
- Verify Appropriateness with Spot Checks
- Iterate on Audience Definition Based on Output
