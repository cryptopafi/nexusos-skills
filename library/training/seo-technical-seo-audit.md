# Technical SEO Audit

[SEO] Technical SEO Audit — Procedura Standard de Operare
Scope: Full technical SEO audit covering crawlability, indexability, Core Web Vitals, structured data, and site architecture to identify and prioritize technical issues blocking organic growth.
Steps:
- Crawl the Site with Screaming Frog or Semrush
- Analyze Core Web Vitals
- Check Indexability, robots.txt, and Sitemap
- Audit Internal Linking Architecture
- Validate Structured Data
- Mobile-First and PageSpeed Analysis
- Fix Prioritization and Remediation Roadmap
