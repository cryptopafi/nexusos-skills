# WordPress Website Setup

[WEBSITE-LANDING-PAGES] WordPress Website Setup — Procedura Standard de Operare
Scope: Configurare completă a unui site WordPress de la hosting până la conectarea cu analytics.
Steps:
- Alege și configurează hostingul
- Instalează WordPress
- Selectează și configurează tema
- Instalează plugin-uri esențiale
- Configurează structura permalink
- Activează SSL și verifică HTTPS
- Creează paginile esențiale
- Conectează Google Analytics
