# Apply the Ask-for-Input Pattern

[AI-PROMPT-ENGINEERING] Apply the Ask-for-Input Pattern — Procedura Standard de Operare
Scope: Instructs the AI to explicitly request specific input before taking any action, preventing premature or misdirected outputs by making all required inputs explicit before proceeding.
Steps:
- Define What Inputs Are Required for the Task
- Instruct the AI to Request Before Acting
- Respond to Each Input Request
- Verify the AI Has Collected All Required Inputs
- Trigger Execution
- Evaluate Input Completeness After Task Completion
