# Sensitivity Analysis (Data Tables and Scenario Manager)

[FINANCE] Sensitivity Analysis (Data Tables and Scenario Manager) — Procedura Standard de Operare
Scope: Construirea și interpretarea analizei de sensitivitate folosind Excel Data Tables și Scenario Manager pentru identificarea driver-ilor cheie de valoare.
Steps:
- Identificarea driver-ilor cheie de valoare
- Construirea tabelului de sensitivitate one-way (1 variabilă vs output)
- Construirea tabelului de sensitivitate two-way (2 variabile vs output)
- Crearea Scenario Manager (Base/Bull/Bear)
- Identificarea variabilelor cu cel mai mare impact (Tornado Chart)
- Documentarea rezultatelor și implicațiilor
