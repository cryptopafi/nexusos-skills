# Detect a Rug Pull or Scam Token

[CRYPTO] Detect a Rug Pull or Scam Token — Procedura Standard de Operare
Scope: Identificarea semnalelor de avertizare ale unui rug pull sau token scam înainte de a investi capital.
Steps:
- Verifică contractul pe Etherscan / BSCScan
- Rulează analiza automată de securitate
- Verifică lichiditatea și lock-ul acesteia
- Analizează distribuția holder-ilor
- Evaluează semnalele sociale și comportamentul echipei
- Testează cu sumă minimă (dacă continui)
- Monitorizează și setează exit rapid
