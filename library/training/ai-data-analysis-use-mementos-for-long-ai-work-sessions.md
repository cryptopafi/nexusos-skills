# Use Mementos for Long AI Work Sessions

[AI-DATA-ANALYSIS] Use Mementos for Long AI Work Sessions — Procedura Standard de Operare
Scope: Menținerea coerenței contextuale prin mementos structurate în sesiuni AI multi-etapă sau multi-zi, eliminând pierderea de progres și re-lucrul inutil.
Steps:
- Define — Definește structura memento-ului pentru proiect
- Create — Creează memento la fiecare pauză de sesiune
- Load — Încarcă memento-ul la reluarea sesiunii
- Verify — Verifică că AI-ul înțelege corect contextul
- Update — Actualizează memento-ul pe parcursul sesiunii
- Archive — Arhivează mementos-urile sesiunilor completate
