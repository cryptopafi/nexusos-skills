# Build a Token Website with AI (Replit)

[CRYPTO] Build a Token Website with AI (Replit) — Procedura Standard de Operare
Scope: Ghid pentru construirea unui website funcțional pentru un token crypto folosind Replit și AI (Claude/GPT), de la structura paginilor până la deployment.
Steps:
- Definește structura și paginile necesare
- Configurează Replit și alege template-ul
- Folosește AI pentru a genera structura HTML/CSS
- Personalizează conținutul și design-ul
- Adaugă secțiunea Disclaimer/Terms obligatorie
- Testează website-ul pe mobile și desktop
- Deployează și configurează domeniul
