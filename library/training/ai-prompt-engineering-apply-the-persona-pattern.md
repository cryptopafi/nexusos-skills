# Apply the Persona Pattern

[AI-PROMPT-ENGINEERING] Apply the Persona Pattern — Procedura Standard de Operare
Scope: Instructs the AI to adopt a specific persona (role, expertise, personality) and maintain it consistently throughout a conversation to produce more contextually appropriate outputs.
Steps:
- Define the Persona
- Anchor the Persona at Conversation Start
- Specify Context and Constraints
- Frame the Request Through the Persona's Lens
- Test Consistency with an Edge Case
- Refine Persona Characteristics Iteratively
- Document the Persona for Reuse
