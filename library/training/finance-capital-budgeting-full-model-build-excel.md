# Capital Budgeting Full Model Build (Excel)

[FINANCE] Capital Budgeting Full Model Build (Excel) — Procedura Standard de Operare
Scope: Construirea unui model complet de capital budgeting în Excel pentru evaluarea unui proiect de investiții, incluzând FCF, NPV/IRR, sensitivity și scenario analysis.
Steps:
- Definire scope proiect și timeline
- Construire proiecții de venituri
- Construire structura de costuri (CapEx + OpEx)
- Calculare Free Cash Flows
- Construire analiză NPV / IRR
- Sensitivity analysis (2-way table)
- Scenario analysis și prezentare stakeholders
