# Apply the Tail Generation Pattern

[AI-PROMPT-ENGINEERING] Apply the Tail Generation Pattern — Procedura Standard de Operare
Scope: Instructs the AI to append relevant follow-up prompts or questions at the end of each response, enabling guided exploration of a topic by surfacing what productive next steps look like from the AI's perspective.
Steps:
- Activate Tail Generation in the Session
- Submit Initial Request and Review the Tail
- Select and Execute a Tail Prompt
- Maintain Thematic Coherence Across Turns
- Collect and Evaluate the Exploration Path
- Save High-Value Tail Prompts as Reusable Templates
