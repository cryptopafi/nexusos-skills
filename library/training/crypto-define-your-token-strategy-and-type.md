# Define Your Token Strategy and Type

[CRYPTO] Define Your Token Strategy and Type — Procedura Standard de Operare
Scope: Cadru metodic pentru definirea strategiei unui token crypto, alegerea tipului potrivit și alinierea cu obiectivele proiectului și cerințele legale.
Steps:
- Definiți problema pe care o rezolvă tokenul
- Alegeți tipul de token potrivit
- Analizați implicațiile legale (Howey Test)
- Definiți token utility și use cases concrete
- Definiți relația token-protocol (token economics basics)
- Evaluați competiția și poziționarea tokenului
- Documentați Token Strategy Document v1.0
