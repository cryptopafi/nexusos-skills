# Google Search Console Setup and Monitoring

[ANALYTICS-TRACKING] Google Search Console Setup and Monitoring — Procedura Standard de Operare
Scope: Configurare și monitorizare Google Search Console — de la verificare proprietate până la monitorizarea Core Web Vitals și alertarea automată pentru erori de indexare.
Steps:
- Adaugă proprietatea în Search Console
- Verifică proprietatea (ownership)
- Trimite sitemap.xml
- Verifică Coverage report pentru erori de indexare
- Analizează Performance report
- Configurează alerte email pentru erori de acoperire
- Monitorizează Core Web Vitals
