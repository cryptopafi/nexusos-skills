# Apply the Template Pattern

[AI-PROMPT-ENGINEERING] Apply the Template Pattern — Procedura Standard de Operare
Scope: Uses a pre-defined structural template with explicit placeholders to constrain AI output format, ensuring consistent structure across multiple outputs of the same type.
Steps:
- Design the Template with Explicit Placeholders
- Document What Fills Each Placeholder
- Provide the Template and Fill Guide to the AI
- Provide the Source Content or Context
- Validate Template Completeness
- Iterate on Template Design
