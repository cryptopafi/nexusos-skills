# Create and Upload Token Metadata (Metaplex/MetaBoss)

[CRYPTO] Create and Upload Token Metadata (Metaplex/MetaBoss) — Procedura Standard de Operare
Scope: Ghid pentru crearea, structurarea și upload-ul metadatelor unui token Solana conform standardului Metaplex Token Metadata, cu hosting pe Arweave sau IPFS pentru persistență permanentă.
Steps:
- Pregătiți assets-urile vizuale ale tokenului
- Structurați fișierul metadata.json conform standardului Metaplex
- Alegeți și configurați soluția de stocare permanentă
- Upload-ați imaginea pe Arweave/IPFS
- Upload-ați fișierul metadata.json
- Atașați metadata URI la Mint Account cu MetaBoss sau Metaplex Sugar
- Verificați integrarea în ecosistem și documentați
