# Buy a Memecoin via DexScreener

[CRYPTO] Buy a Memecoin via DexScreener — Procedura Standard de Operare
Scope: Procesul de identificare, verificare și achiziție a unui memecoin pe DEX folosind DexScreener ca instrument principal de analiză.
Steps:
- Găsește token-ul pe DexScreener și obține adresa contractului
- Aplică verificările C-005 (rug pull detection)
- Verifică metricele cheie pe DexScreener
- Configurează wallet-ul și verifică slippage
- Execută cumpărarea cu sumă strict limitată
- Setează target de exit și stop mental
- Monitorizează și execută exit conform planului
