# Apply the Question Refinement Pattern

[AI-PROMPT-ENGINEERING] Apply the Question Refinement Pattern — Procedura Standard de Operare
Scope: Uses the AI to surface ambiguities in an initial prompt by generating targeted clarifying questions, then rewrites the prompt with those answers incorporated to produce a significantly higher-quality final output.
Steps:
- Write and Submit the Initial Prompt
- Review the Clarifying Questions
- Answer the Category A and B Questions
- Request Prompt Rewrite
- Approve or Correct the Refined Prompt
- Generate Final Output from Refined Prompt
