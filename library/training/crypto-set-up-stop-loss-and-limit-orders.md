# Set Up Stop-Loss and Limit Orders

[CRYPTO] Set Up Stop-Loss and Limit Orders — Procedura Standard de Operare
Scope: Configurarea corectă a ordinelor stop-loss și limit pe exchange pentru a gestiona riscul și a automatiza execuția tranzacțiilor.
Steps:
- Calculează riscul acceptat per trade
- Identifică nivelul tehnic de stop-loss
- Calculează dimensiunea poziției
- Deschide poziția și plasează imediat stop-loss
- Configurează Limit Order pentru target de profit
- Monitorizează și ajustează (trailing stop)
- Documentează tranzacția
