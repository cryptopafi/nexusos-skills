# Tesla-Style Full Financial Model Build

[FINANCE] Tesla-Style Full Financial Model Build — Procedura Standard de Operare
Scope: Construirea unui model financiar complet pentru o companie cu multiple segmente de venituri și CapEx intensiv, folosind Tesla ca studiu de caz paradigmatic.
Steps:
- Cercetarea companiei — Business model și unit economics
- Construirea modelului de livrări de vehicule
- Construirea modelului de venituri din Energy și Services
- Modelarea marjelor gross per segment
- Construirea modelului de cheltuieli operaționale
- Construirea modelului CapEx (factories și charging)
- Build 3-Statement Model complet
- Valorizare DCF cu scenarii de creștere
