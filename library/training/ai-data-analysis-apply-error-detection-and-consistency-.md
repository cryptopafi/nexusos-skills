# Apply Error Detection and Consistency Checking

[AI-DATA-ANALYSIS] Apply Error Detection and Consistency Checking — Procedura Standard de Operare
Scope: Verificarea sistematică a output-urilor AI pentru erori factuale, logice, de formatare și hallucinations, asigurând calitatea și consistența livrabilelor.
Steps:
- Define — Definește tipologia erorilor relevante pentru contextul specific
- Setup — Construiește verificatorul de consistență
- Run Consistency — Verifică consistența internă a output-ului
- Cross-reference — Verifică faptele față de surse externe
- Flag — Cataloghează toate discrepanțele identificate
- Correct and Re-validate — Corectează și re-validează
