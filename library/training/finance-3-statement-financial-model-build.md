# 3-Statement Financial Model Build

[FINANCE] 3-Statement Financial Model Build — Procedura Standard de Operare
Scope: Construirea unui model financiar integrat (Income Statement + Balance Sheet + Cash Flow Statement) cu cele trei declarații legate și verificate intern.
Steps:
- Construirea tabului de Assumptions
- Construirea Income Statement
- Construirea Balance Sheet
- Construirea Cash Flow Statement
- Linkarea celor trei situații
- Verificarea că Bilanțul se echilibrează
- Adăugarea Scenario Manager
