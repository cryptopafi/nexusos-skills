# Run a Crypto Livestream Campaign

[CRYPTO] Run a Crypto Livestream Campaign — Procedura Standard de Operare
Scope: Planificarea și execuția unui livestream crypto pentru promovarea unui token sau proiect, de la setup tehnic până la follow-up post-eveniment.
Steps:
- Definește obiectivul și formatul livestream-ului
- Promovează evenimentul cu minim 48h înainte
- Pregătește setup-ul tehnic
- Pregătește agenda și materialele vizuale
- Execută livestream-ul conform agendei
- Exportă și redistribuie înregistrarea
- Compilează metrics și raportul post-stream
