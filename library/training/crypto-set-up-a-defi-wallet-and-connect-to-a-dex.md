# Set Up a DeFi Wallet and Connect to a DEX

[CRYPTO] Set Up a DeFi Wallet and Connect to a DEX — Procedura Standard de Operare
Scope: Configurarea unui wallet self-custody pentru DeFi și conectarea sigură la un DEX pentru prima tranzacție.
Steps:
- Instalează wallet-ul din sursa oficială
- Creează wallet-ul și securizează seed phrase-ul
- Adaugă rețeaua și token-urile necesare
- Finanțează wallet-ul și verifică adresa
- Conectează-te la DEX prin URL oficial (bookmark imediat)
- Execută primul swap cu sumă minimă de test
- Gestionează și revocă aprobările de contract
