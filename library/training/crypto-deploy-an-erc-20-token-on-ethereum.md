# Deploy an ERC-20 Token on Ethereum

[CRYPTO] Deploy an ERC-20 Token on Ethereum — Procedura Standard de Operare
Scope: Ghid complet pentru scrierea, testarea și deployarea unui contract ERC-20 pe rețeaua Ethereum, inclusiv verificare pe Etherscan.
Steps:
- Pregătire environment de dezvoltare
- Scrieți contractul ERC-20
- Compilați și inspectați ABI-ul
- Scrieți și rulați testele
- Deploy pe testnet Sepolia
- Verificați contractul pe Etherscan
- Deploy pe mainnet și documentare finală
