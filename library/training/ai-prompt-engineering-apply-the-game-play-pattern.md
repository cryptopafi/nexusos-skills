# Apply the Game Play Pattern

[AI-PROMPT-ENGINEERING] Apply the Game Play Pattern — Procedura Standard de Operare
Scope: Structures an AI interaction as a game with defined rules, goals, scoring, and progression mechanics to elicit higher-quality outputs through structured constraint and engagement.
Steps:
- Define the Game Structure
- Present Game Rules to the AI
- Execute Round 1
- Apply Scoring and Enforce Rules
- Increase Challenge with Level Progression
- End Game and Extract Outputs
- Post-Game Debrief
