# Apply the ReAct Prompting Pattern

[AI-PROMPT-ENGINEERING] Apply the ReAct Prompting Pattern — Procedura Standard de Operare
Scope: Implements the Reasoning + Acting (ReAct) loop for complex multi-step tasks by interleaving explicit thought steps with actions that gather information or transform state, producing traceable and verifiable reasoning chains grounded in real observations.
Steps:
- Define the Task and Available Actions
- Initialize the ReAct Loop
- Provide Observations for Each Action
- Monitor the Reasoning Chain for Quality
- Trigger Final Answer Only When Sufficient Observations Exist
- Verify the Final Answer Traces to Observations
