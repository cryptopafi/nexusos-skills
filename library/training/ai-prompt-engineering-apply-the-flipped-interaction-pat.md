# Apply the Flipped Interaction Pattern

[AI-PROMPT-ENGINEERING] Apply the Flipped Interaction Pattern — Procedura Standard de Operare
Scope: Inverts the standard prompting dynamic so the AI asks you targeted questions to gather all necessary information before generating output, replacing a poorly-specified one-shot prompt with a structured interview.
Steps:
- Define the Goal and Activate Interview Mode
- Respond to Each Question Concretely
- Signal Interview Completion
- Review the AI's Summary of Your Inputs
- Approve and Trigger Output Generation
- Evaluate Interview Efficiency
