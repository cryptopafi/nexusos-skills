# Fraud Prediction Model Building

[FINANCE] Fraud Prediction Model Building — Procedura Standard de Operare
Scope: Construirea end-to-end a unui model de machine learning pentru detectarea fraudei financiare, de la feature engineering la deployment și monitorizare.
Steps:
- Colectarea și înțelegerea dataset-ului etichetat
- Feature Engineering — variabile predictive
- Split train/test (80/20) cu stratificare
- Antrenarea modelului — Random Forest sau XGBoost
- Evaluarea modelului — precision/recall/AUC-ROC
- Setarea threshold-ului pe baza toleranței businessului
- Implementarea în pipeline de producție și monitorizarea drift-ului lunar
