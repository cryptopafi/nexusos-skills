# Apply the Semantic Filter Pattern

[AI-PROMPT-ENGINEERING] Apply the Semantic Filter Pattern — Procedura Standard de Operare
Scope: Applies semantic criteria (tone, sentiment, relevance, audience-appropriateness, or other meaning-based criteria) as explicit filters on AI-generated output, ensuring content meets qualitative standards that cannot be enforced by simple keyword rules.
Steps:
- Define the Semantic Filter Criteria
- Apply the Filter at Generation Time
- Request a Post-Generation Self-Audit
- Review Flagged Sections
- Regenerate Flagged Sections
- Calibrate Filter Sensitivity
