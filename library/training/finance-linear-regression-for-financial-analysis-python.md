# Linear Regression for Financial Analysis (Python)

[FINANCE] Linear Regression for Financial Analysis (Python) — Procedura Standard de Operare
Scope: Aplicarea regresiei liniare OLS pentru estimarea beta-ului acțiunilor, analiza relației return-factor, și interpretarea riguroasă a rezultatelor statistice.
Steps:
- Definirea variabilelor dependente și independente
- Colectarea și pregătirea datelor
- Rularea regresiei OLS cu statsmodels
- Interpretarea coeficienților — beta/alpha, R², p-values, t-stats
- Construirea graficului regresiei cu scatter și linia de regresie
- Verificarea rezidurilor — normalitate și heteroscedasticity
- Aplicarea insight-urilor din regresie
