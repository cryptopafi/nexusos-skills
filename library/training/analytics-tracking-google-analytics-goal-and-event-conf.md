# Google Analytics Goal and Event Configuration

[ANALYTICS-TRACKING] Google Analytics Goal and Event Configuration — Procedura Standard de Operare
Scope: Configurare completă a goal-urilor și event-urilor în GA4 — de la definirea conversiilor până la testarea cu DebugView și monitorizarea săptămânală.
Steps:
- Definește goal-urile de conversie
- Creează events custom în GA4 pentru fiecare goal
- Marchează events cheie ca conversii
- Configurează trigger-ele în GTM
- Testează events cu GA4 DebugView
- Creează rapoarte de goal completion
- Monitorizează ratele de conversie săptămânal
