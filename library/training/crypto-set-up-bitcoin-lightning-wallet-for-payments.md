# Set Up Bitcoin Lightning Wallet for Payments

[CRYPTO] Set Up Bitcoin Lightning Wallet for Payments — Procedura Standard de Operare
Scope: Ghid pas-cu-pas pentru configurarea și utilizarea unui Bitcoin Lightning Network wallet pentru plăți rapide și cu comisioane mici.
Steps:
- Alege wallet-ul Lightning potrivit cazului tău de utilizare
- Instalează și configurează wallet-ul
- Înțelege și configurează canalele Lightning (pentru wallet-uri non-custodial)
- Finanțează wallet-ul cu Bitcoin
- Generează și testează o invoice Lightning
- Configurează backup-ul și recuperarea
- Testează scenariul complet de plată și documentează în Cortex
