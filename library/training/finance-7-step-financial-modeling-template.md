# 7-Step Financial Modeling Template

[FINANCE] 7-Step Financial Modeling Template — Procedura Standard de Operare
Scope: Construirea unui model financiar complet și prezentabil în 7 pași standardizați, de la definirea scopului la output-ul final pentru decizie.
Steps:
- Definire scop și output-uri
- Tab Assumptions — toate input-urile clar etichetate
- Tab Historical Financials (3-5 ani)
- Tab Revenue Model (pe segment/produs)
- Tab Cost Model (fix vs. variabil)
- Three-Statement Model integrat (IS/BS/CF)
- Tab Output (DCF / Comps / Sensitivity) și formatare finală
