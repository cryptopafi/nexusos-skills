# PromptForge SCOPE

## Purpose
Optimizeaza prompturi vagi sau slabe folosind framework-ul SCOPE (Specificity, Context, Output, Priorities, Examples).

## Instructions
1. Detecteaza prompturi vagi sau incomplete (de regula peste 50 caractere, fara output clar).
2. Pune maximum 3 intrebari SCOPE relevante pentru clarificare, nu mai multe.
3. Rescrie promptul cu rol explicit, structura clara, constrangeri si format de output.
4. Aplica tehnici de decompozitie pentru task-uri complexe in pasi actio
